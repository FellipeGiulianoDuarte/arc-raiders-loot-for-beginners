# Arc Raiders Loot Guide for Beginners

Visual loot decision guide for Arc Raiders beginners. Helps you decide what to keep, sell, recycle, and prioritize during raids.

**Live site:** https://fellipegiulianoduarte.github.io/arc-raiders-loot-for-beginners/

## Sections

- **Loot Guide** — 15 key items with detailed info (where to find, crafting uses, tips). Click any item for full details.
- **Workshop** — Upgrade requirements for all 7 stations with RVI (Recycling Value Index) to decide what to do with extras.
- **Expedition** — Material requirements for Expedition Project 1, all 5 phases.
- **Quests** — 20 items to keep for quest progression.
- **Golden Rules** — 7 rules every beginner should follow.

## Data Sources
